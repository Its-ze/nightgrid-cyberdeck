InstallersLinux + Windows

Linux One-Line Install

Downloads the latest AppImage, marks it executable, and creates a desktop launcher.

curl -fsSL https://its-ze.github.io/nightgrid-cyberdeck/install-linux.sh | bash

Windows One-Line Install

Downloads the latest Windows setup executable and launches the installer.

irm https://its-ze.github.io/nightgrid-cyberdeck/install-windows.ps1 | iex

USB Notes

On Linux, add your user to the serial group if ports cannot open. Meshtastic CLI commands need exclusive port access.

sudo usermod -aG dialout "$USER"
NightGrid runs locally. Serial traffic, mesh output, and GPS coordinates stay on your machine unless you copy or send them.
