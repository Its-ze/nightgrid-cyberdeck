Linux Install / Update

Downloads the latest AppImage, uses a writable per-user fallback if needed, installs the launcher icon, and creates a managed Meshtastic CLI venv for Mesh commands.

curl -fsSL https://its-ze.github.io/nightgrid-cyberdeck/install-linux.sh | bash

Windows Install / Update

Downloads a fresh Windows setup executable and launches the installer/updater.

irm https://its-ze.github.io/nightgrid-cyberdeck/install-windows.ps1 | iex

USB Notes

On Linux, add your user to the serial group if ports cannot open. Meshtastic firmware logs are line-buffered and stripped of ANSI color codes.

sudo usermod -aG dialout "$USER"

Heltec V3 War Drive

Continuously polls the selected Heltec V3 or Meshtastic receiver while you move, requires GPS by default, stamps node sightings with cached USB GPS fixes, and saves local JSONL/CSV logs.

Start / Poll now / Require GPS / Save log

Linux Deb Lid Mode

The deb package installs a NightGrid-owned systemd-logind drop-in so your mobile deck keeps running with the laptop lid closed, then removes it on package removal.

/etc/systemd/logind.conf.d/nightgrid-cyberdeck-lid.conf

Command Deck

Macro buttons, richer status tiles, RX/TX/status log filters, copy logs, clear logs, and selected-device quick actions.

Selected help / Mesh nodes / GPS push / ESP32 ping

Marketplace

Tabbed pack browser with Featured, All, T-Deck, T-Dongle, ESP32, Smart, Firmware, and Lab lanes plus Source, Grab, Auto setup, and Run actions.

WLED / ESPHome / CircuitPython / ESP-NOW / Matter

ESP32 Module

Auto-connects generic ESP32 serial modules and exposes reset, bootloader, Ctrl-C/Ctrl-D, help, file-list, remote-control, and T-Deck Link actions.

CP210x / CH340 / Espressif USB serial

ESP32 Remote

Links a selected ESP32 to the computer, then sends identify, heartbeat, Wi-Fi scan, I2C scan, GPIO write, and custom commands over REPL or JSON Link.

Link host / Identify / GPIO write / Run

Flasher

Targets T-Deck, ESP32-S3, ESP32, and T-Dongle serial boards with public web flasher links plus connect, reset, and bootloader controls.

Open flasher / Firmware / Bootloader / Reset

T-Deck Link

Sends CyberDeck Link probe, pair, deck-ready, launcher, and text JSON events through the selected ESP32 serial module.

{"cmd":"attachProbe","deckId":"itsz-tdeck"}

T-Dongle Bridge

Uses the CyberDeck Link USB serial protocol to buffer fragmented attach heartbeats, show USB and pairing status, check the T-Dongle GUI, open Wi-Fi settings, write the bridge URL, start/finish pairing, queue remote payloads, and send text/control events to a T-Deck.

{"cmd":"attachProbe","deckId":"itsz-tdeck","deckName":"ITSZ T-Deck"}

T-Deck Remote Link

Guided Check GUI, Wi-Fi settings, Start link, and Finish link controls distinguish USB-alive from Wi-Fi-unjoined, keep split dongle JSON out of the RX log, report failed Command Deck actions correctly, and wake the T-Deck launcher.

CyberDeck-Link / http://192.168.4.1

GPS Probe / Push

Pick a USB GPS serial port, probe for live NMEA sentences at the selected baud, connect the parser, then push cached fixes to the T-Dongle bridge.

{"cmd":"text","text":"GPS 38.897957,-77.036560 sat=9"}

Linux Uninstall

Removes the AppImage from standard and fallback locations, plus the launcher and installed deb package if present. App settings are preserved unless you opt into purge.

curl -fsSL https://its-ze.github.io/nightgrid-cyberdeck/uninstall-linux.sh | bash

Windows Uninstall

Finds the registered NightGrid uninstaller and runs it, then cleans standard shortcuts.

irm https://its-ze.github.io/nightgrid-cyberdeck/uninstall-windows.ps1 | iex
NightGrid runs locally. Serial traffic, mesh output, War Drive logs, and GPS coordinates stay on your machine unless you copy, send, save, or push them.
